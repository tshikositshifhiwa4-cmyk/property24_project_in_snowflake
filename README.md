## 📌 Overview

This project analyzes Property24 real estate data using Snowflake, transforming raw listings into meaningful insights on pricing, location trends, and property characteristics.

---

## 🎯 Objectives

* Clean and structure raw property data
* Analyze pricing across provinces and cities
* Identify key drivers of property value
* Answer real-world business questions using SQL

---

## 🛠️ Tech Stack

* **Snowflake** – Cloud data warehouse
* **SQL** – Data transformation & analysis
* **VS Code** – Development

---

## 📂 Dataset

Includes:

* City & Province
* Property Price
* Bedrooms & Parking
* Floor Size
* Monthly Repayment

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset into Snowflake

### 2. Data Cleaning

* Handled nulls and inconsistencies
* Standardized formats

### 3. Data Analysis

* Price filtering & ranking
* Provincial comparisons
* Affordability analysis

Based on structured SQL questions 

---

## 📊 Key Insights

* Gauteng & Western Cape have higher property prices
* Property size strongly affects price
* Some regions offer more affordable housing
* Monthly repayment aligns with price trends
